## ✨ About Project

MyFinance is a personal finance management website built as part of my learning journey in web development using PHP and MySQL.

The application helps users manage their income, expenses, savings, investments, and financial reports through a simple, modern, and responsive interface.

> 🚧 **This repository is currently under active development. New features and improvements will be added continuously.**

---

## 🚀 Features

### ✅ Available Features

* 📊 Responsive dashboard
* 💰 Income management
* 💸 Expense management
* 🔍 Search transactions
* 🌙 Dark and light mode
* 📱 Responsive interface
* 💾 MySQL database integration
* 🗑️ Delete transactions with confirmation dialog

### 🚧 Upcoming Features

* ✏️ Edit transactions
* 🎯 Savings goals and progress tracking
* 📈 Investment goals and tracking
* 🔔 Debt due notifications
* 📅 Calendar and date filters
* 📑 Weekly reports
* 📑 Monthly reports
* 📊 Financial charts and analytics
